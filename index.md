## 🔥 About This Project

This is a **hands-on enterprise systems administration lab** designed to simulate real-world IT infrastructure environments.

✔ Built using Proxmox VE  
✔ Configured Active Directory Domain  
✔ Integrated Wazuh SIEM  
✔ Implemented Backup & Disaster Recovery  
✔ Simulated Security Incidents  

---

## ⚡ Proven Capabilities

- Built enterprise AD domain (DC01 + WIN11)
- Configured DNS, DHCP, authentication
- Deployed SIEM monitoring (Wazuh)
- Implemented backup & restore strategy
- Simulated real-world incidents
- Troubleshot system failures
